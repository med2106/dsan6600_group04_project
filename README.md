# dsan6600_group04_project
DSAN 6600 - Neural Networks Final Project

To install `uv`, follow the instructions on the `uv` GitHub page. For example, on Mac, you can run

```
curl -LsSf https://astral.sh/uv/install.sh | sh
```

To install `poetry`, follow the [instructions](https://python-poetry.org/docs/) on the `poetry` documentation. On my machine, I ran the following commands

```
brew install pipx
pipx install poetry
```

To install env:

```
uv sync
source .venv/bin/activate
```

Add Git LFS for large data files

```
sudo apt-get install git-lfs
git lfs install
```
